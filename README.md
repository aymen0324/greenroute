# 🌐 GreenRoute - Plataforma de Logística Sostenible

## 📋 Descripción del Proyecto

GreenRoute es una plataforma de logística verde de última generación que combina inteligencia artificial, optimización de rutas y sostenibilidad ambiental. Desarrollada con tecnologías modernas para ofrecer una experiencia de usuario excepcional y funcionalidades avanzadas de gestión logística.

## 👨‍💻 Desarrollado por

**Aymane El Khilaly**  
Estudiante de Informática - Universidad Miguel Hernández (UMH)

## 🚀 Características Principales

- **Diseño Ultra-Moderno 2025**: Interfaz futurista con glassmorphism y animaciones fluidas
- **Optimización de Rutas con IA**: Algoritmos avanzados para rutas eficientes
- **Dashboard en Tiempo Real**: Métricas y análisis instantáneos
- **Arquitectura Responsive**: Experiencia perfecta en todos los dispositivos
- **Navegación Multi-página**: Sistema de enrutamiento optimizado
- **Integración Blockchain**: Transparencia en la cadena de suministro

## 🛠️ Stack Tecnológico

### Frontend
- **React 18** - Framework principal
- **TypeScript** - Tipado estático
- **Vite** - Build tool moderno
- **Framer Motion** - Animaciones avanzadas
- **Tailwind CSS** - Styling utility-first
- **React Router** - Navegación SPA
- **Recharts** - Gráficos interactivos
- **Zustand** - Gestión de estado
- **React Query** - Data fetching
- **React Hook Form** - Formularios
- **Zod** - Validación de esquemas

### Testing & Quality
- **Vitest** - Testing framework moderno
- **React Testing Library** - Testing de componentes
- **ESLint** - Linting de código
- **Prettier** - Formateo de código
- **Husky** - Git hooks
- **lint-staged** - Pre-commit hooks

### Desarrollo
- **PostCSS** - Procesamiento CSS
- **Autoprefixer** - Compatibilidad CSS

## 📦 Instalación

```bash
# Clonar el repositorio
git clone https://github.com/aymen0324/greenroute.git

# Navegar al directorio
cd greenroute

# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run dev

# Construir para producción
npm run build
```

## 🚀 Scripts Disponibles

```bash
# Desarrollo
npm run dev          # Servidor de desarrollo
npm run build        # Construir para producción
npm run preview      # Vista previa de producción

# Testing & Quality
npm run test         # Ejecutar tests
npm run test:ui      # Tests con interfaz visual
npm run test:coverage # Tests con cobertura
npm run lint         # Linting de código
npm run lint:fix     # Linting con auto-fix
npm run type-check   # Verificación de tipos
npm run format       # Formateo de código

# Análisis
npm run analyze      # Análisis del bundle

# Deploy
npm run deploy:vercel    # Deploy a Vercel
npm run deploy:netlify   # Deploy a Netlify
npm run deploy:gh-pages  # Deploy a GitHub Pages
```
